# 🦄 Unicorn Rainbow Dash

Ein magisches Einhorn-Flugspiel, gebaut mit Vibe-Coding! 🌈✨

## Spielen

👉 **[Jetzt spielen](https://sabahattingiritli.github.io/ELIZ/)**

Oder lokal: Öffne einfach `index.html` in einem Browser.

## Steuerung

| Aktion | Gerät |
|---|---|
| Fliegen | Leertaste oder Mausklick |
| Fliegen | Tippen (Handy/Tablet) |

## Features

- 🌈 **Regenbogen-Schweif** mit magischem Glitzer-Effekt
- ⭐ **Sterne sammeln** für Punkte (doppelt im Turbo!)
- ⛈️ **Gewitterwolken** als Hindernisse
- 💎 **Glitzer-Diamant** aktiviert den Rainbow-Dash-Turbo!
- ☀️🌙 **Tag-Nacht-Zyklus** mit Glow-Effekten
- 🎵 **Prozedurale Magik-Musik** via Web Audio API
- 📱 **Handy-optimiert** (Touch + Querformat)
- 🏆 **Highscore** wird lokal gespeichert

## Anpassen (Vibe-Config)

Öffnet `vibe-config.js` und ändert einfach die Werte:

```js
gravity: 0.25,        // Wie schwer ist das Einhorn?
lift: -6,             // Wie hoch springt es?
obstacleFrequency: 0.003, // Wie viele Wolken?
turboDuration: 6,     // Wie lange hält der Turbo?
```

## Technologien

- Vanilla HTML + CSS + JavaScript
- Web Audio API (keine externe Bibliothek!)
- Canvas 2D API
